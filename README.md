# Amazon Product Page Clone

A front-end clone of an Amazon product page, built as part of a web development course project.

## 🔗 Live Demo
[View Live Page](https://andrawsHaj.github.io/Amazon-Project)

## 📌 About The Project
This project is a pixel-perfect recreation of an Amazon product page for the **MSI Katana A15 AI Gaming Laptop**. The goal was to replicate the layout, design, and functionality of a real Amazon product page using only HTML and CSS.

## 🛠️ Technologies Used
- **HTML5** — Semantic structure
- **CSS3** — Styling and layout
- **Font Awesome** — Icons

## ✨ Features
- **Top Navigation Bar** — Logo, search bar, account links and cart
- **Bottom Navigation Bar** — Category links
- **Sponsored Banner** — MSI promotional banner
- **Breadcrumb Navigation** — Page location trail
- **Product Image Gallery** — Thumbnail selector using pure CSS radio inputs, with sticky scroll behavior
- **Product Details** — Title, rating, Amazon's Choice badge, price, CPU/RAM/Storage options, specs table, and about section
- **Buy Box** — Price, shipping info, Add to Cart and Buy Now buttons
- **Sponsored Product Card** — Related product advertisement
- **Customers Frequently Viewed** — Horizontal product carousel
- **Products Related to This Item** — Sponsored products carousel
- **Responsive Design** — Scales down to fit mobile screens (iPhone SE)
